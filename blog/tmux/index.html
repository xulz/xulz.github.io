利器之TMUX

时间： 2017-12-18   |   分类： tech     |   阅读： 492 字 ~1分钟

用法

  • tmux ls : list sessions
  • tmux attach -t 0 : -t 代表“target” , 0 是session名
  • tmux kill-session -t 0 : 结束会话
  • tmux source ~/.tmux.conf : 重新加载配置
  • 复制和粘贴: 使用Shift临时禁用tmux,然后Ctrl+Shift+C

常用快捷键

Key Description
C-b (Ctrl-b) Send the prefix key
M (Alt, meta key)
prefix c new-window
prefix “ split-window
prefix % split-window -h
prefix z resize-pane -Z
prefix $ Rename the current session
prefix , Rename the current window

prefix M-5 select-layout tiled

使用会话保存和恢复插件

我精彩使用自动保存和恢复的功能，用到了插件管理和几个插件。 使用步骤如下：

# 1. 安装 Plugin Manager
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
# 2. 编辑 .tmux.conf 或.tmux.conf.local
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-resurrect'
set -g @plugin 'tmux-plugins/tmux-continuum'
set -g @continuum-restore 'on'
run '~/.tmux/plugins/tpm/tpm'

# 3. 安装配置里的插件
prefix + I # 大写的I代表Install
prefix + U # 更新插件

# 4. 重启后恢复会话
prefix + Ctrl-s # 保存
prefix + Ctrl-r # 恢复

小技巧

技巧1: 总是保留一个会话

终端启动时先运行 tmux attach -t base || tmux new -s base

技巧2: 多面板命令同步

把当前输入发送到当前窗口的所有面板 setw synchronise-panes

技巧3: 缩放

prefix + z放大当前面板, 再次输入恢复。
