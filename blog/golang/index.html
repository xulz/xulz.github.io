Go语言学习笔记

时间： 2017-12-26   |   分类： Tech     |   阅读： 0 字 ~0分钟

接触Go有段时间了，感觉有几个学习资源很有帮助：

  1. 官方Go指南 : 边写边学, 最轻松简单的方式
  2. 系统的学习: 推荐书 ， 我是先看的中文电子版之后买的实体书。

  3. 官方文档”Effective Go/实效Go编程” : 理解Go的必读文档

    自己的体会是如果只看不写，等写代码时会发现实际没记住多少。这些好资料有的地方需要多看几遍，再结合平时多写代码去理解。

下面是一些Go语言学习笔记，作为学习的汇总。

new和make函数

new返回指针,make返回值

  • new(T) : 为类型T的新对象申请零值内存并返回内存地址,(不初始化内存)的好处是不需要初始化函数.

  • make(T, args) ： 为slice,map,channel等返回初始化类型T后的值(非指针)

type Thing struct {
    Name  string
    Num   int
}
// 如果包里只有一个类型，函数名可以简写为New
// 构造函数使用的简写格式
func NewThing(someParameter string) *Thing {
    return &Thing{someParameter, 33}
}
